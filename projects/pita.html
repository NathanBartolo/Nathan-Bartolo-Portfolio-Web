Pita Restaurant Website

Project Type Website
HTML CSS Web Design Branding

A visually rich restaurant website designed based on the client’s desires as part of a group project for a Filipino restaurant in Dubai. The site captures the warmth and authenticity of Mediterranean cuisine, featuring an appetising visual hierarchy, clear menu showcases, and a brand-consistent layout. It also includes practical features such as an ordering system that directs orders straight to the restaurant’s contacts, creating a seamless and engaging dining experience for visitors.
