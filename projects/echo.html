Echo Website

Project Type Website
React Node.js Full-Stack Music MERN API Integration MongoDB

Echo is a full-stack music streaming platform built using the MERN stack (MongoDB, Express.js, React, and Node.js), designed to deliver a seamless and immersive listening experience. It features secure user authentication, playlist creation and management, and the ability to browse, search, and like tracks.

The platform integrates the iTunes API for dynamic music data and supports Google API login for quick and secure user access. With a sleek dark-themed interface, it incorporates smooth animations, dynamic transitions, and interactive UI elements—creating an engaging and intuitive user experience.
